# Transportation Management System (C++)

## Description
A console-based Transportation Management System built using C++ and OOP concepts.  
Manages rides, drivers, and passengers with features to add, update, search, and display entities.

## Features
- Add, update, and display drivers
- Add, update, and display passengers
- Add rides linking drivers and passengers
- Console-based interface
- Demonstrates OOP concepts: classes, inheritance, encapsulation

## Tech Stack
- C++ (OOP)
- Standard Template Library (STL)
